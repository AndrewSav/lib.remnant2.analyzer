Dataset
  List<Character>: Index (could be different from screen order if a char was deleted), ActiveWorldSlot (enum:Campaign/Adventure)
    Profile: Inventory, MissingItems, HasMatsItems (e.g. can get right now), HasWormhole (for Archon), HasFortuneHunter (for Archon)
    SaveSlot: Campaign (RolledWorld), Adventure (RolledWorld), QuestCompletedLog (for Oracle's Quilted Heart), HasTree (for Ripened Heart)
      RolledWorld: QuestInventory, Ward13 (generated), AllZones = Zones + Ward13
        List<Zone>: Name
          List<Location>: Name, WorldStones, Connections, TraitBook, Simulacrum, WorldDrops, DropReferenes, Category(maps to world/biome)
            List<LootGroup>: Name(for vendor, overworld POI, boss, miniboss, injectable), Type (boss, injectable, miniboss, overworld POI, location, dungeon, vendor, world drop), EventDropReference
              List<LootItem>
